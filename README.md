# 🚆 Train Consist Management App

## 📌 Overview

The **Train Consist Management App** is a system designed to manage train compositions (consists), including coaches, engines, and their arrangement. It helps in efficiently organizing, updating, and tracking train configurations.

---

## 🎯 Features

* ➕ Add new train consist
* 🚃 Manage coaches and engines
* 🔄 Update train composition
* ❌ Remove coaches or trains
* 📋 View complete train details
* ⚡ Efficient data handling using optimized logic

---
